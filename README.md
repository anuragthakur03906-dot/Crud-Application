# MERN CRUD Application

A full-stack CRUD (Create, Read, Update, Delete) application built with the MERN stack (MongoDB, Express.js, React.js, Node.js) and Tailwind CSS.

## Features

- **Create** - Add new users with form validation
- **Read** - View all users in a beautiful responsive table
- **Update** - Edit existing user information
- **Delete** - Remove users with confirmation dialog
- **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **Form Validation** - Using Formik & Yup for robust validation
- **Modern UI** - Tailwind CSS with beautiful green gradient theme
- **Sequential IDs** - Auto-numbered (1, 2, 3...) like the image reference

## Tech Stack

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework for REST APIs
- **MongoDB** - NoSQL database
- **Mongoose** - ODM for MongoDB data modeling

### Frontend
- **React.js** - Frontend library
- **Vite** - Fast build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Formik** - Form management
- **Yup** - Schema validation
- **Axios** - HTTP client for API calls

## Prerequisites

Before running this project, make sure you have:

- Node.js (v14 or higher) installed
- MongoDB installed locally or MongoDB Atlas account
- npm or yarn package manager
